--- layout: default title: Datasets --- We maintain a collection of reference datasets which have been formatted following CF conventions. Please note that while we work to keep this collection up to date, much of the data has been first downloaded from primary sources and then reformatted for use in our system. Please check the global attributes of the files for reference information. This collection is available for download via the ILAMB software system. After installing the ILAMB python package, you can use the ilamb-fetch utility to download or update our collection of reference data.

Identifying and adding new datasets is a continual effort which we coordinate via the ILAMB-Data Github repository. Please consider submitting an issue if you have a suggestion.
